# Лабораторная 4. Отсечение отрезков

Отсечение (clipping) – это процедура обнаружения и исключения из дальнейшего анализа элементов сцены (линий, поверхностей объектов), которые не попадают в видимую область, отображаемую на экране.

## Алгоритм средней точки

Рассмотрим прямоугольное окно. Каждую область плоскости, которая получается разбиением эого окна - пометим 4-х битным кодом, каждый бит которой будет обозначать принадлежность одной из координат области (например x > x_max)

1. Если отрезок по длине меньше пикселя - не рассматриваем
2. Если отрезок полностью не лежит в области - не рассматриваем
3. Если отрезок полностью в окне - отобразить
4. Разделить отрезок пополам и вновь применить алгоритм для каждого из них

Недостатки:

1. Большая глубина рекурсии

Преимущества:

1. Может хорошо подойти для области определенной выпуклым многоугольником (не прямоугольником)

## Алгоритм средней точки для выпуклого многоугольника

Аналогично прямоугольной области. Но отличие только в том что здесь не получится быстро определять расположение точки относительно многоугольника
